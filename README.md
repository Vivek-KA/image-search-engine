  Image Search Web

A simple and responsive Image Search Website built using HTML, CSS, and JavaScript, powered by the Unsplash API.
Users can search for images by keyword and view high-quality photos dynamically loaded from Unsplash.

• Technologies Used:

   HTML5 – Structure of the website
, CSS3 – Styling and layout
, JavaScript (ES6) – Logic, DOM manipulation, async/await
, Unsplash API – Fetching images

• Features:
   Search images by keyword
, Fetches real-time images from Unsplash
, Responsive image grid layout
, Opens images in a new tab
, Clean and simple UI
, Uses async/await for API calls
